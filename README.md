# gitignore

```
gitignore [<options>] [project] [path]

USAGE:
    gitignore <project> [path]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <project>    gitignore를 생성할 project
    <path>       지정한 경로에 .gitignore 생성 [default: .]
```
